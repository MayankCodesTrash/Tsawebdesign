# Echoes of the Fort

A community story website capturing the history, heroes, and heart of Des Moines, Iowa.

**TSA Website Design 2026 — Community Story Challenge**

## Quick Start

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Deploy

```bash
npm run build
```

Upload the `dist` folder to Vercel, Netlify, or any static host.

## Pages

- **Home** — Landing page with overview
- **History** — Interactive timeline (1843–2026)
- **Heroes** — Notable figures archive + quiz
- **Education** — Metro school districts
- **City Builder** — Interactive incremental game
